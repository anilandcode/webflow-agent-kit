# Claude Desktop MCP Example

Add this to your `claude_desktop_config.json` to give Claude Desktop access to Webflow.

## Setup

1. Copy the `config.json` content into your `claude_desktop_config.json`
2. Replace `<YOUR_TOKEN>` with your Webflow API token
3. Restart Claude Desktop

Claude will now have access to all Webflow tools through the MCP protocol.
