# @webflow-agent-kit/cli

CLI runner for webflow-agent-kit.

## Install

```bash
npm install -g @webflow-agent-kit/cli
```

## Usage

```bash
wfak auth       # Check auth status
wfak sites      # List sites
wfak tools      # List available tools
```

See the [main README](https://github.com/anilandcode/webflow-agent-kit) for full documentation.
