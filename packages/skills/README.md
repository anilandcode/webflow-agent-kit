# @webflow-agent-kit/skills

Manifest-backed safe workflow packs for webflow-agent-kit.

## Install

```bash
npm install @webflow-agent-kit/core @webflow-agent-kit/skills
```

## Usage

```typescript
import { createWebflowAgentKit } from '@webflow-agent-kit/core';
import { SkillExecutor } from '@webflow-agent-kit/skills';

// See skill-packs/ for documented skill recipes
```

See the [main README](https://github.com/anilandcode/webflow-agent-kit) for full documentation.
